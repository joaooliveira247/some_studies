# Let's install git.

## [Linux](https://git-scm.com/download/linux)

### Debian/Ubuntu

```bash
sudo apt install git
```

### [Windows](https://git-scm.com/download/win)

### Using winget

```bash
winget install --id Git.Git -e --source winget
```

### [MacOS](https://git-scm.com/download/mac)

### Using brew

```bash
brew install git
```
